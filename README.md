Make sure you have installed git and node 
Clone the application by running the following command
git clone 

Navigate to the project directory using cd command

Install dependencies 
command : npm install 
create .env and set value of the port as indicated in env.dev file 

Run the application by 
command:npm run dev 

To test application : user swagger documentation which 
http://localhost:{port}/api-docs/

Make sure you replace the port by the one you used in .env
